## forkify

### App Description

A food recipe web-app where you can search recipes, add ingredients to a shopping list, and like your favorite recipes.

### Technologies Used

- Webpack
- Babel
- JavaScript with ES6 Syntax
- HTML5
- CSS3 w/ Mobile First Responsive Design
- AJAX
- NPM
